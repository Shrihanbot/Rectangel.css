Rectangel.css

Rectangel is a CSS library designed with simplicity in mind. Since many people use CSS libraries with tons and tons of classes in their HTML, this can make it hard to understand, and the user ends up using it improperly. Because of this, Rectangel has very easy to understand classes. Stay tuned for updates!
